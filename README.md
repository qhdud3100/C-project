## 필수 
``` 
git clone [프로젝트 주소]

git init 
git remote add [원격 repo 이름] [github 주소] \n

git branch [branch 이름]
git checkout [branch 이름]
git branch -b [삭제할 branch 이름]

git add [file 이름]
git commit -m "메세지"

git push [원격 repo 이름] [branch 이름]
```

``` 
git remote -v
git status 
git branch
```
